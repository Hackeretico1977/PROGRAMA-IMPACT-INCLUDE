## 5. Gestión de Identidad Digital y Certificados
En este apartado recojo mi experiencia personal con la identidad digital. Tengo configurados y utilizo todos los certificados principales (excepto los de pago):

### Mis Certificados y Sistemas de Acceso
* **FNMT:** Certificado que mas utilizo que dura 4 años y se instala en el explorador Edge,Chrome o firefox.  https://www.fnmt.es/
* **Sistemas de la AOC (Consorci Administració Oberta de Catalunya):**  https://idcat.aoc.cat/
    * **idCAT Mòbil:** Sistema de identificación basado en el envío de contraseñas de un solo uso al móvil. No todas las web de organismos aceptan este sistema.No caduca
    * **idCAT Certificat:** Certificado digital en software para trámites con administraciones catalanas y estatales. caduca cada 4 años
* **Registro Cl@ve:** Sistema de Clave PIN y Clave Permanente para acceso a la Seguridad Social, Hacienda y otros organismos igual que el resto, se obtiene o el de clave permamente o pin
* **DNI electrónico:** El sisema de obtención y renovación es diferente como cada uno de ellos pero sirven para lo mismo.https://www.dnielectronico.es/PortalDNIe/
* **Camefirma este certificado se tramita desde su web, dura 2 años y en este caso es e pago para ciudadno persona fisica són 35e
* **Programa Autofirma, no funciona solo, sino que en muchas administraciones que piden la firma atraves de  autofirma el funcionamiento es se  abre el programa se busca el documento a firmar se seleciona el certificado y se firma.

### Proceso de renovación del DNIe
El pin y certificados caducan cada dos años, el proceso de renovación a seguir es:
1. **Acudir a Comisaría:** Es necesario el trámite presencial en una oficina de la Policía Nacional.
2. **Máquinas PAD:** Uso de los Puntos de Actualización del DNIe sin necesidad de cita previa.
3. **Validación Biométrica:** Se introduce el DNI y se confirma la identidad con la huella dactilar para renovar los certificados y el PIN al momento.
4. Importante si no esta caducado solo puedes renovar la contraseña sin alterar la caducudad original.
